# NuMicro_M5531_BSP

**Nuvoton NuMaker-M5531 Board Support Pack (BSP)**, a CMSIS software pack, is designed to work with all compiler toolchains (Arm Compiler, GCC, IAR). It is released as [CMSIS software pack](https://www.keil.arm.com/packs/numicro_m5531_bsp-nuvoton) and therefore accessible by CMSIS-Pack enabled software development tools.


## Top-level folder structure

- Documents<br>
    Documentations provided by Nuvoton.

- Examples<br>
    Blinky example using CMSIS VIO driver.

- Images<br>
    Pictures of the board.

- Library<br>
    CMSIS VIO driver.
